# 🐾 Animais Fantásticos

Aplicação web desenvolvida com foco em **JavaScript moderno (ES6+)**, explorando conceitos como **modularização, manipulação do DOM, consumo de API e automação de build com Webpack**.

O projeto foi inicialmente baseado no curso de JavaScript da Origamid, porém evoluído com **implementações próprias**, melhorias de UX e **setup profissional de build (Webpack + Babel)**.

---

## 🚀 Funcionalidades

* Navegação dinâmica entre seções
* Manipulação e atualização do DOM em tempo real
* Consumo de API com `fetch`
* Uso de `async/await`
* Atualização automática de dados
* Sistema de **notificações (toast)**
* **Alternância de tema (light/dark mode)**
* Estrutura modular de código
* Layout responsivo
* Animações e transições com JavaScript

---

## ⚙️ Setup moderno (Webpack + Babel)

O projeto utiliza um ambiente de build profissional:

* **Webpack** para empacotamento dos módulos
* **Babel** para compatibilidade com diferentes navegadores
* Geração de bundle em `/dist`
* Scripts automatizados para desenvolvimento e produção

---

## 📡 API Mock

O projeto utiliza um arquivo local (`animaisapi.json`) como mock de API para simular requisições HTTP durante o desenvolvimento, eliminando dependência de serviços externos.

## ▶️ Como executar o projeto

```bash

```

```bash

```

### 🚀 Build de produção

```bash
npm run build
```

# 🐾 Animais Fantásticos

Projeto desenvolvido com **HTML, CSS e JavaScript**, com foco em **manipulação do DOM, modularização de scripts, consumo de API e interatividade na interface**.

Este projeto foi desenvolvido com base nos **estudos e práticas do curso de JavaScript da Origamid**, onde foram aplicados conceitos fundamentais para o desenvolvimento front-end moderno, como **funções assíncronas, eventos, animações, troca de tema e atualização dinâmica de conteúdo**.

Além da base proposta no curso, foram adicionadas **melhorias e funcionalidades extras implementadas por mim**, como animações toast, switch de tema claro/escuro e atualização automática de dados via API.

---

## 🚀 Funcionalidades

* Navegação interativa entre seções
* Manipulação dinâmica do DOM
* Consumo de API com `fetch`
* Uso de `async/await`
* Atualização automática de dados
* **Toast animations para feedback visual ao usuário**
* **Switch para alternância entre tema claro e escuro**
* Estrutura modular de scripts
* Layout responsivo
* Animações e interações com JavaScript

---

## 🛠️ Tecnologias utilizadas

* **Fetch API**
